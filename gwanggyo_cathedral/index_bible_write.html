Google Apps Script 웹 앱 URL을 입력하면 모든 기록이 스프레드시트에 자동 저장됩니다.
설정 방법:
  1. Google Sheets를 열고 확장 프로그램 → Apps Script를 선택합니다
  2. 제공된 Code.gs 내용을 붙여넣고 저장합니다
  3. 배포 → 새 배포를 클릭, 유형: 웹 앱, 액세스: 모든 사용자로 설정합니다
  4. 생성된 웹 앱 URL을 아래에 붙여넣고 연결합니다
