## ℹ️ 소개
저는 **학생들의 성장을 돕는 고등학교 교사**이자, **사용자 경험을 고민하는 웹 개발자**입니다. 

학교에서는 학생들에게 '웹 프로그래밍'과 '데이터베이스 프로그래밍'을 가르치며 탄탄한 IT 기본기를 다져주는 데 집중하고 있습니다. 개인적으로는 **프론트엔드 개발과 UI/UX 디자인**에 깊은 열정을 가지고 있으며, 복잡한 데이터를 직관적이고 아름다운 인터페이스로 풀어내는 과정에서 큰 보람을 느낍니다. 

주로 최신 프론트엔드 트렌드를 연구하여 사용자 친화적인 웹 애플리케이션을 개발하는 데 몰두하며, 새로 배운 기술들을 학생들을 위한 교육 자료나 토이 프로젝트에 적용해 보는 것을 즐깁니다.

## 🛠️ 기술 스택
- **프로그래밍 언어**: JavaScript, TypeScript, HTML/CSS, SQL
- **프레임워크/라이브러리**: React, Next.js, Tailwind CSS
- **도구**: Git, VS Code, Figma (UI/UX 기획)
- **기타**: REST API, MySQL/MariaDB

## 🚀 포트폴리오
- **[인터랙티브 웹 학습 플랫폼](#)**: 학생들이 웹 프로그래밍 기초를 직접 타이핑하며 배울 수 있도록 React로 구현한 에듀테크 웹사이트입니다. 직관적인 UI를 통해 학습 몰입도를 높였습니다.
- **[학교 동아리 홍보 페이지](#)**: UI/UX 기획부터 프론트엔드 개발까지 전담하여 제작한 반응형 웹사이트로, 모바일과 데스크톱 모두에서 매끄러운 사용자 경험을 제공합니다.
- **[학생 포트폴리오 관리 DB 시스템](#)**: 데이터베이스 프로그래밍 수업과 연계하여 설계한 프로젝트로, 효율적인 데이터 스키마 설계 및 SQL 쿼리 최적화를 적용했습니다.
